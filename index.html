What it does

Here is an overview of all things the app can do for you

Freesker Lite

Core diagnostics and reporting tools


Dashboard

A concise summary of your vehicle’s health, showing active faults and recent activity — designed for quick insights and straightforward diagnostics

Version Check

Verify ECU software versions and compare them to the expected vehicle configuration. Ensure firmware compatibility across modules

Fault Check + DTC Report via PDF

Read and clear Diagnostic Trouble Codes (DTCs) from ECUs or the full network. Export detected faults into a structured PDF for documentation or remote analysis




Freesker Pro

Advanced configuration and control tools


Basic Functions

Soft Reset

Controlled ECU restart to recover from temporary faults without changing configuration

Hard Reset

Complete ECU reset to restore operational state for persistent issues

Tester Alive

Keeps diagnostic session active and confirms ECU communication during scripts




Advanced Functions

ECU Configuration Setup

Write and verify base parameters for ECU operation after replacements or updates

VCU Enable / Disable

Temporarily isolate or reset high-voltage systems during maintenance

TBOX Reset

Restore telematics box to default for OTA and remote communication recovery

Close Hand Brake

Apply the electronic parking brake for diagnostic or safety procedures

Release Hand Brake

Release the electronic parking brake to allow service work such as brake bleeding or wheel removal

MRR Write Role Configuration

Assign or rewrite the physical role of each mid-range radar (Front Center, Front Left, Front Right, Rear Left, Rear Right)

Front Camera Calibration Setup

Reinitialize or overwrite front camera calibration data to restore ADAS alignment and correct “Calibration Out of Range” errors

Train Doggy Windows

Perform a complete training/calibration cycle for the doggy window function to ensure proper sensor behavior and automatic window operation

Add NFC Cards

Register and pair new NFC access cards with the vehicle’s system

Check NFC Cards

View the number and status of all NFC cards currently paired

Delete NFC Cards

Remove one or multiple NFC cards from the vehicle’s paired list

Add Key Fob

Pair a new key fob to the vehicle for keyless access and start authorization

Number of Paired Key Fobs

Display the current count of key fobs recognized by the vehicle

Delete Key Fob

Unpair an existing key fob from the system

How to get started

It's so easy, your mom could do it

Get the Dongle

To be able to connect to your car, buy the Vgate vLinker FD+ Dongle from our recommended suppliers or Amazon.

Setup Car + Browser

Connect the dongle to your car and set it ready. Use a Web compatible browser - Chrome on Mac/Android or Microsoft Edge.

Start the App

Run the app any time on your mac, windows or android, let it connect to your car and enjoy all available tools.
