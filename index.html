What it does

Here is an overview of all things the app can do for you.

  • Version Check
    Our Version Check diagnostic feature performs a thorough health check of your vehicle's software by verifying the software version of each Electronic Control Unit (ECU) against the latest standards, including OS 2.2, 2.1, and 2.0; it identifies any components that are not up to date and provides detailed insights on required updates, such as recommending upgrades from OS 2.0 or OS 2.1 to OS 2.2 for optimal performance and reliability.
  • Fault Check
    Our Fault Check feature thoroughly examines your vehicle‚Äôs systems using advanced diagnostic algorithms to identify potential faults. With comprehensive reporting, you receive insights into detected issues, and logs can be generated to provide technicians with remote access to the vehicle's condition for further analysis.
  • Train Doggy Windows
    Are you experiencing issues with your Doggy Windows not functioning properly? Our diagnostic software can help resolve these problems by recalibrating the window systems to ensure they operate as intended.
  • Add/Remove/Check NFC card
    dd new cards by following on-screen prompts, or remove them through the NFC settings to ensure authorized access only.Check the status of your registered cards anytime for peace of mind. With this feature, enjoy enhanced security and effortless access to your Fisker Ocean, making every journey more convenient.
  • Add Key Fob
    With the Freesker Tool, you can easily add Key Fobs for enhanced vehicle access and convenience. Simply follow the on-screen prompts in your vehicle‚Äôs interface to register key fobs, allowing you and your family to unlock and start your vehicle effortlessly.
  • Hard reset of T-box
    The T-Box Hard Reset is a comprehensive system restoration process. It completely erases all user data and settings from the device. The process reinstalls the original system software and resets all configurations to their default values.
  • Number of paired Key Fobs
    This feature enhances vehicle security through robust key authentication protocols. The system employs advanced encryption technology to secure communication between the vehicle and key fobs. Additionally, this function allows users to determine the exact number of key fobs currently paired with the vehicle, providing a clear overview of active access points.
  • Check NFC card
    The Function Check NFC Card feature verifies the functionality of the vehicle's Near Field Communication (NFC) system. It tests the NFC reader's ability to detect and read NFC-enabled cards or devices.
  • Calibrate Radar
    Don't settle for less than perfect. With our Radar Calibration feature, you're investing in peace of mind and the future of your driving experience. It's not just a function - it's your personal guardian on the road!

How to get started

Its so easy, your mom could do it.

Get the Dongle
To be able to connect to your car, buy the vGate vLinker FD+ Dongle ie from here but may also be available on Amazon and other stores.
Setup Car + Browser

FAQ & Support

We are here to help. Get in touch to get support.

When will the details in the imprint be available?
The contact details and imprint will be published upon the registration of the company. We are currently working on the establishment of Freesker.
What is the status of the company formation?
The formation of Freesker is in the final phase. We will keep you updated on the progress and the official opening.
How will my pre-order be confirmed?
Pre-orders will be confirmed individually, as we want to ensure all information is accurate. Therefore, please note that confirmations for pre-orders may arrive late. Conspicuous data will result in a block, but this can be lifted by submitting a support request.
When will the final order be triggered?
After the registration of the company, you will receive an email with a payment link. This email will be sent to all customers who have placed a pre-order.
How does payment for the final order work?
Please ensure that you make your payment by credit card within the specified timeframe to secure your claim to the respective discount offer.
How are my data stored and protected?
All diagnostic-relevant data are retrieved directly from the server in the online version. Your VIN is used to unlock the demo mode. No other customer or vehicle data is stored on our server to ensure maximum data security.
When will the offline version be available?
We are currently planning an offline version of our tool, which is expected to be usable 12 months after purchase. This allows us to ensure a fully functional product while maximizing data security.
Which features will be included in the full version?
Following features are planned for the full version:
- Clear DTCs (Diagnostic Trouble Codes)
- Recalibrate lane assist
- Recalibrate radars (ADAS)
- Recalibrate seat heating
- Force update key fobs and door handles
- Enable/disable HV system
- Soft reset ECU
- Hard reset ECU
Further features will be added continuously.
Can I use the software for multiple vehicles?
No, the license is tied to a single VIN (Vehicle Identification Number) and cannot be shared between vehicles. However, we can provide a customized fleet license for multiple vehicles. To proceed, certain proof of ownership for the respective vehicles must be submitted to our support team.
What happens to the license if the vehicle is sold?
The license remains associated with the VIN and can be transferred along with the vehicle. The new owner can request re-assignment through Freesker.
How do I select the correct server if the vehicle was originally sold in the US but is now in Europe?
The server should be selected based on the vehicle's current location. In this case, the EU server should be used.
Will the software work on any Android tablet?
Yes, it will work as long as the connection instructions for pairing the dongle with the device are followed. For more details, visit:
US Dashboard EU Dashboard
Is the vLinker FD+ required, or are there compatible alternatives?
Only the vLinker FD+ is fully tested with the software, other dongles are not supported officially.
How do I set up the 6-digit PIN during the first login?
The first time you use the tool you will be asked to enter a 6 digit PIN. Please remember this for future use of Freesker with this vehicle. The PIN can be reset with a support request.
The traing doggy window function does not work. What can I do?
1. Ensure you are logged into the correct server.
2. Check for a stable network connection.
3. Turn off your browser‚Äôs automatic translation feature.
Will it be possible to recalibrate the Lane Keeping Assistant with the tool after release?
Yes, recalibration and repair of the lane-keeping assistant will be possible with the full version.
How can I resolve browser compatibility issues for the demo version?
1. Use a supported browser such as Google Chrome or Microsoft Edge.
2. If you‚Äôre using Google Chrome, follow these steps to enable "Experimental Web Platform Features":
1. Launch the Google Chrome browser.
2. Type chrome://flags into the address bar.
3. Hit Enter to access the Flags menu.
4. Search for "Experimental Web Platform features".
5. Set the drop-down menu from Default to Enabled.
6 .Restart your browser.
3. After these steps, the demo version should work as expected.
